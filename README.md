## üöÄ Key Features

### 1. Neural Consensus Engine (Federated Learning)
-   **Zero-Trust**: Raw data never leaves the on-premise node.
-   **Gradient Sync**: Transmits only `JSON` vectors representing model weight adjustments.
-   **Trust Scoring**: Gamified "Reputation System" penalizes nodes that submit poor-quality data (preventing data poisoning).

### 2. Real-Time "Circuit Breakers"
-   **Velocity Checks**: Automatically halts simulation pipelines if transaction frequency exceeds 99th percentile.
-   **Batch Attack Simulation**: Admin tools to simulate 20+ concurrent botnet threads to test system resilience.

### 3. Graph Intelligence & Visualization
-   **Entity Knowledge Graph**: Visualizes "hidden" connections between seemingly unrelated wallets using simulated Graph Neural Net (GNN) logic.
-   **Risk Heatmaps**: Real-time aggregation of global threat levels.

## üì¶ Getting Started

### Prerequisites
-   Node.js v18+
-   npm v9+

### Installation

1.  **Clone the Architecture**
    ```bash
    git clone https://github.com/sentinel-core/fraud-intel.git
    cd FraudIntel
    ```

2.  **Install Dependencies (Monorepo)**
    ```bash
    cd client && npm install
    cd ../server && npm install
    ```

3.  **Launch the Grid**
    ```bash
    # Terminal 1: Start the Federation Server
    cd server
    node index.js
    
    # Terminal 2: Start the Dashboard
    cd client
    npm run dev
    ```

4.  **Access the Nodes**
    -   **Landing**: `http://localhost:3000`
    -   **Bank A Node**: `http://localhost:3000/dashboard/BANK_A`
    -   **Federation Hub**: `http://localhost:3000/hub`
